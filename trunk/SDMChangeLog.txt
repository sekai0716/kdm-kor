Version 1.0
	기초적이었던 MFC 버전으로써 아주 기본만 담고 있던 프로그램

Version 1.1
	KDM이 GNU 오픈소스이기 때문에 마음대로 수정 및 재배포가 가능하여
	KDM 1.5.5버전을 개조하여, SDM 1.1버전을 만듦.

	기능 추가: 도트데미지 및 정령의 소환수 공격 데미지도 계산
	기능 추가: 치유성의 신성한 기운, 정령의 기운류 데미지도 계산
	기능 추가: 메뉴 및 폼 한글화
	기능 추가: 툴팁에 몇가지 기능 추가
	기능 추가: 프로그램 아이콘 변경

	기능 추가: 레지스트리 읽어서 자동으로 Chat.log 파일 읽어옴

	기능 추가: 프로그램 실행시 chat.log파일의 내용을 지워줌

Version 1.1.6
	기능 추가: 윈도우 비스타, 7의 경우 관리자 권한으로 실행 관련 안내 메시지 보강
	기능 추가: 살성, 호법, 궁성의 독 바르기, 폭약 바르기, 독 화살, 중독의 덫, 폭발의 덫, 바람의 약속 데미지 집계 가능

	기능 추가: 파티원이 정령성의 불길의 축복으로 데미지 발생시 정령성의 데미지에 합산

	버그 수정: 정령이 떄린 치명타 데미지만 정령의 평타 데미지로 합산됬던 버그 수정
	버그 수정: 마도성의 화산 폭발 및 지연 폭발 스킬의 데미지 관련 버그 수정
